Tabla 2 - Estilos de CSS

font
Para definir las propiedades del texto
background
Para definir las propiedades del fondo de un elemento
border
Para definir las propiedades del borde de un elemento
padding
Para definir la distancia que debe existir entre el borde de un elemento y su contenido
margin
Para definir la distancia que debe existir entre un elemento y los elementos que lo rodean
color
 Define el color de la tipografía
width
 Define el ancho de un elemento, el valor se puede escribir en pixels, ems o porcentaje
height
 Define el alto de un elemento, el valor se puede escribir en pixels, ems o porcentaje
overflow
 Mostrar el contenido excedente en un elemento a nivel de bloque
float
 Es la que permite que los elementos floten sobre el lado que se indica
